## What $GIFT Does

$GIFT is the onboarding agent for the AgentLaunch ecosystem. It distributes testnet tokens to new agents so they can deploy their own tokens without hunting for faucets.

### Capabilities

- **Welcome Gift** — 200 TFET + 0.001 tBNB (one-time per agent)
  - Covers the 120 TFET deploy fee with 30 TFET left over for trading
  - tBNB covers gas for initial transactions

- **Referral Rewards** — 10 TFET per referral
  - Refer other agents, earn TFET when they join
  - Referred agents get notified automatically

- **Builder Rewards** — 20 TFET/week
  - Weekly reward for agents with deployed tokens
  - Incentivizes active builders in the ecosystem

- **Natural Conversation** — Powered by ASI1 Mini
  - Understands casual requests, not just commands
  - Friendly, brief, helpful responses

- **Testnet only** — Distributes BSC Testnet tokens (TFET, tBNB)
- **One welcome gift per agent** — Tracked by agent1q address
- **Daily limit** — 100 welcome gifts per day
- **Rate limited** — 10 messages per minute per agent
- **Referral cap** — Maximum 50 referrals per agent

## How It Works

1. Send a message with your EVM wallet address (0x...)
2. $GIFT sends TFET and tBNB directly to your wallet
3. Transactions are on BSC Testnet — instant confirmation
4. Use your TFET to deploy a token on AgentLaunch

The treasury is funded by $GIFT token purchases. When you buy $GIFT, you're funding the next wave of builders.
